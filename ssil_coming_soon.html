Mission / Vision / Value

Mission

私たちの使命・存在意義

地域の固有資産と先端技術を融合し、社会的価値と経済利益が循環する持続可能な未来を創る

Vision

私たちが目指す未来の姿

相模から世界へ。地域とデジタルが響き合い、次世代の産業と人材が育ち続けるエコシステムの創成

Value

私たちが共有する行動指針・価値観（CSV Driven）

1. 社会的価値と経済的価値の両立: 共通価値の創造 (CSV Driven)
2. 今ある持ち札からすぐ始める: 手中の鳥の原則 (Bird in Hand)
3. 仲間や地域、サポーターとの対話による共創: クレイジーキルトの原則 (Crazy Quilt)
4. 小さく試して、賢く転ぶ: 許容可能な損失の原則 (Affordable Loss)

Core Businesses

単なる慈善活動にとどまらず、市場のニーズに応える強固なビジネスモデルを通じて社会的課題を解決する「CSV (Creating Shared Value)」アプローチにより、以下の4つのコア事業を有機的に連携させて展開します。

01

地域経済活性化・産業創出事業（総論・プラットフォーム）

相模原市を中心とした神奈川県中西部エリアをフィールドとし、地域の未利用資源や潜在能力を掘り起こします。将来的には海外販路開拓やグローバル市場とのマッチングを通じて外貨を獲得し、地域へ還元する次世代の持続可能な産業エコシステムを創出・育成します。

02

アグリビジネス事業・グローバル産業創出（かながわ農業版MBA）

農業を独立したビジネスとして再定義し、「儲かる農業」への転換を志向。持続可能なアグリビジネスを自社・パートナー・地域と共に創出すると同時に、海外の最先端アグリテックや輸出戦略に精通した次世代の高度な農業人材を育成します。（創業当初は佐野川茶を中心とした足柄茶の事業創出、農業人材の育成、および観光事業に注力）

03

伝統産業次世代継承・国際教育事業（エデュテイク）

地域に根差す伝統産業や文化的価値を「探究学習」などの教育コンテンツと結びつけ、次世代のファンを育てる教育事業を展開。インバウンド層や海外の教育機関向けの国際文化教育プログラム、および越境取引へと発展させ、地域の文化的資産を世界的な価値へと昇華させます。

04

シビックテック・デジタル利活用事業（コア・基盤）

市民参加型の社会課題解決（シビックテック）をベースに、AIや先端テクノロジーなどのデジタル技術を最適に利活用する事業を創出。海外の先進的なコミュニティや国際的な技術スタンダードと連携し、各事業へ横断的に注入することで、全事業のスケールと社会的インパクトを世界水準へと引き上げます。

Action Principles

不確実性の高い環境に適応し、迅速に行動するため、私たちは以下の「エフェクチュエーション（起業家的意思決定理論）」の5つの原則を共有し、実践します。

手中の鳥の原則 (Bird in Hand)
まずメンバーの現在の持ち札・リソース（専門性、技術、人脈）を最大限に活かして、今できる具体的な行動から着手する。
許容可能な損失の原則 (Affordable Loss)
予測できない将来の利益を最大化しようとするのではなく、万が一失敗した場合でも各自が許容できる損失（時間、エネルギー、出資資金など）を予め見定め、その範囲内で検証と挑戦を行う。
クレイジーキルトの原則 (Crazy Quilt)
社会的ミッションに共感し、コミットしてくれる多様なパートナー（顧客、地域住民、他企業、外部サポーター等）との協働関係を早期に構築し、彼らとの対話を通じて事業の輪郭を共創していく。
レモネードの原則 (Lemonade)
予期せぬ事態や計画の失敗、偶然の出来事をリスクとして排除せず、新たな事業の機会やヒントとして捉え、柔軟に方向転換を行う原動力とする。
飛行機のパイロットの原則 (Pilot in Plane)
予測不能な未来をコントロールしようとするのではなく、自分たちが「今コントロールできる要素」に集中し、自らの手で未来を創り出す。
